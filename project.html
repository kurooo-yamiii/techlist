TechList: Automated Inventory System

The TechList: Automated Inventory System has a user-friendly interface, that the user can easily navigate the system. This system allows the administrator to manipulate the inventory system; it includes the add, delete and update of data inside the database.

FEATURE

REAL-TIME
The generated inventory system is using an online database making the input and outputted information updated for all the users of the application, when the administrator of the application deletes, add or update data it will reflect constantly to the user without any traffics and disturbances.
SEARCH ENGINE
Proponents implement the search bar with a combination of modifiers to track data & information within the table list. Every item inventory has its own search bar which also included the logs and manage accounts the search results will be listed within the table in ascending order.
SCHEDULING
This is considered as a new feature of the TechList having the projector of the administrator prone to borrowing the proponents create the scheduling and log system in order to monitor the use of the projector and its status whether its available for borrowing or defective.
MULTI-LOGIN
Having two users of the application including the users and admin the proponents included the feature of a multi-login system in the application. In order to limit and restrict manipulation the data level of users is issued, users can’t alter data on the other hand administrator can manipulate data.

UPDATES

TechList Patch 1.0.1 : December, 2022
Redefinition of designs to make the label highlighted for the clarity of the users, changing the icon background color to dark orange. Forms are also labeled through the title of the form removing excess labels in different form to avoid confusion hence the organization of the buttons is newly arranged.
TechList Patch 1.0.2 : January 2023
Emphasis on database security, the database is more likely to be altered with the use of text boxes and symbols therefore in this patch unauthorized characters are banned. Clicking or typing an unauthorized character on the application will trigger the notification box and delete the typed character.
TechList Patch 1.0.3 : February 2023
Excluding projector inventory into the main inventory, the computer in the laboratory became the main inventory also by double-clicking the computer it will go directly to the computer parts. Adding add, delete and update for both computer parts and projector inventory.
TechList Patch 1.0.4 : March 2023
Additional feature for the TechList, borrowing of projectors in the previous patch the projector is excluded hence the proponents add some scheduling to the projector having the status “Available” and “Used”. Proponents take advantage of the real-time feature of the application thus scheduling is added to the application function.
TechList Patch 1.0.5 : April 2023
Equipment and tools inventory became the main inventory having the projector, computer and its parts there is some equipment in the laboratory that is not covered hence the proponents added the equipment and tools inventory to maximize exponentially the inventory system in the computer laboratory.
TechList Patch 1.0.6 : May 2023
Manual system of sending reports to automatic sending of reports, having the three main branches of the inventory including the equipment & tools, projector, computer, and parts “Send Report” button has been implemented. Instead of typing the report manually, the use can instantly select the item to report and directly send it.
